# zrok-auto-install
Script to install zrok docker version for self-hosting on Linux machine

To install zrok-docker type this in your Ubuntu machine's terminal and proceed with steps:

`bash <(curl -fsSL https://raw.githubusercontent.com/kazekagyeee/zrok-auto-install/main/install-ubuntu.sh)`

To install zrok-docker type this in your Debian machine's terminal and proceed with steps:

`bash <(curl -fsSL https://raw.githubusercontent.com/kazekagyeee/zrok-auto-install/main/install-debian.sh)`
